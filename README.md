# 2048 React App

A simple 2048 game built with React and Typescript.

## Installation

1. Clone the Repository
2. cd 2048ReactApp
3. npm install
4. npm start

## How to Play

Use arrow keys (Up, Down, Left, Right) or 'w', 's', 'a', 'd' to move the tiles.
Combine tiles with the same number to reach 2048 and win the game.
